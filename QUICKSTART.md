# Quickstart

## 1. Run an API shell

### Integrated shell
```bash
make run-integrated
```

### Stateful shell
```bash
make run-stateful-phase2
```

## 2. Run tests
```bash
make test-api
```

## 3. Check routes quickly
```bash
make check-routes
```

## 4. Open browser routes
- `/product-home`
- `/operating-center`
- `/project-control-plane`
- `/discovery`
- `/conversion`
- `/validation`
- `/workspace`
- `/stateful-lab`
- `/stateful-actions`
- `/demo-readiness`

## 5. Best demo path
1. Open `/product-home`
2. Move to `/operating-center`
3. Walk the core modules
4. Finish with `/stateful-lab` and `/stateful-actions`
